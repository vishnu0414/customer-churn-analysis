# Customer Churn Analysis

## Overview
This project contains a Power BI report (`customer_churn_analysis.pbix`) for analyzing customer churn in a telecommunications or subscription-based service. The report visualizes key metrics and insights to help understand factors contributing to customer retention and churn.

## Data Sources
The analysis is based on customer data including:
- **Training Dataset**: `archive/customer_churn_dataset-training-master.csv`


### Dataset Features
- **CustomerID**: Unique identifier for each customer
- **Age**: Age of the customer
- **Gender**: Gender of the customer (Male/Female)
- **Tenure**: Number of months the customer has been with the service
- **Usage Frequency**: How often the customer uses the service
- **Support Calls**: Number of support calls made by the customer
- **Payment Delay**: Number of days the customer is late on payments
- **Subscription Type**: Type of subscription (Basic, Standard, Premium)
- **Contract Length**: Length of the contract (Monthly, Quarterly, Annual)
- **Total Spend**: Total amount spent by the customer
- **Last Interaction**: Days since last interaction with the customer
- **Churn**: Target variable indicating if the customer churned (1 = Churned, 0 = Retained)

## Requirements
- Power BI Desktop (latest version recommended)
- Access to the .pbix file and associated CSV datasets

## How to Use
1. Download and install Power BI Desktop from the official Microsoft website.
2. Open the `customer_churn_analysis.pbix` file in Power BI Desktop.
3. If prompted, connect to the CSV data sources in the `customer_churn_dataset-training-master.csv`.
4. Explore the visualizations and insights in the report.
5. Interact with filters and slicers to analyze different segments of the data.

## Key Insights
The report may include analyses on:
- Churn rates by demographics (age, gender)
- Impact of tenure and usage on retention
- Relationship between support calls and churn
- Financial metrics like total spend and payment delays
- Subscription and contract preferences
