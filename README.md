# 📌 Overview

The **Customer Churn Analysis Dashboard** is an interactive **Power BI** report that helps analyze customer behavior and identify factors contributing to customer churn.

The dashboard enables users to monitor customer retention, evaluate subscription performance, and discover actionable business insights using interactive reports, slicers, and KPI cards.

---

# ✨ Dashboard Features

| Feature | Description |
|---------|-------------|
| 📊 KPI Cards | Customer metrics at a glance |
| 📈 Churn Trend Analysis | Understand churn patterns |
| 👥 Customer Segmentation | Analyze customer groups |
| 💰 Revenue Analysis | Spending behaviour |
| 📞 Support Call Analysis | Customer support insights |
| 📅 Contract Analysis | Compare contract types |
| 🎛 Interactive Filters | Dynamic slicers |
| 📉 Drill Down Reports | Explore detailed data |
